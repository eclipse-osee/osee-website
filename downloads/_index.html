Install Instructions

OSEE consists of a standard eclipse update site and an application server.
  • Client Install Instructions:
    See Eclipse Installation on how to install using this update site.
    When launching the client you must set the following java system properties:
    -Dosee.authentication.protocol=trustAll
    -Dosee.arbitration.server=http://<serveraddress>:<serverport>
     The server port must match the server's org.osgi.service.http.port system property. In the server launch files provided, the port number is set to 8089 Refer to OSEE Configuration Properties for further information on setting these properties.
  • Server Instructions:
    1. Unzip the archive to a convenient location on the server machine (might be the same as the client).
    2. Launch using the startup files provided
Refer to OSEE Configuration Properties for further information on setting these properties.
