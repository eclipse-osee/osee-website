Install Instructions

OSEE Client (Update-Site)

  1. Follow Eclipse Update Site Installation instructions.
  2. Set the following java system properties before launching:
    -Dosee.authentication.protocol=trustAll
    -Dosee.arbitration.server=http://<serveraddress>:<serverport>
    The server port must match the server's org.osgi.service.http.port system property. In the server launch files provided, the port number is set to 8089
    Refer to OSEE Configuration Properties for further information on java system property settings.

OSEE Application Server

  1. Unzip the archive to a convenient location on the server machine (might be the same as the client).
  2. Launch using the launch files provided (*.sh, *.bat)
    Refer to OSEE Configuration Properties for further information on java system property settings.
